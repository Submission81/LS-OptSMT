# LS-OptSMT (MaxSMT)

## Build

Compile the project using make:

```bash
make
```

After compilation, the executable `LS-OptSMT` will be generated.

## Clean

Clean the compiled files:

```bash
make clean
```

## Testing

After compilation, you can test using the following commands:

```bash
./LS-OptSMT ./test/hard.smt ./test/soft.smt2
```
